# Repaso corto para entrevistas

## Preguntas que puedo defender

1. Que problema resuelve Docker frente a correr una app directo en el host.
2. Por que Docker Compose ayuda en una app con backend, frontend y base de datos.
3. Que hace un pipeline CI/CD y que diferencia hay entre CI y CD.
4. Por que no se suben secrets al repositorio.
5. Para que sirve un healthcheck.
6. Que diferencia hay entre Deployment, Pod y Service en Kubernetes.
7. Por que un PVC es importante para una base de datos.
8. Que problema resuelve Ingress.
9. Que aporta Helm frente a manifiestos YAML sueltos.
10. Que guarda Terraform en el state.
11. Por que el state no se debe commitear.
12. Como se escala una app con Kubernetes y con Terraform.
13. Que mire en Prometheus y Grafana.
14. Como verifico que una app esta realmente conectada a la DB.
15. Que cambiaria si tuviera que pasar esto a produccion.
